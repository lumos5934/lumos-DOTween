## ℹ️기능

### Tween Preset
`Create / [ LumosLib ] / Scriptable Objects / Tween Preset / `

<img width="295" height="382" alt="image" src="https://github.com/user-attachments/assets/72d7d492-1bbd-4a6b-bd2f-0bb15902af91" />

<br>

자주 사용되는 유형의 애니메이션들을 SO 를 이용해 프리셋화. 현재는 Scale , Fade 에 대한 두가지 이며 지속적으로 필요에 따라 기본적인 프리셋은 추가 예정. 프리셋은 기본적으로 수치만 전달하는 컨테이너 역할을 수행함.

### Tween Animator

<img width="298" height="173" alt="image" src="https://github.com/user-attachments/assets/b6fa8592-13a6-4464-a30f-c758d5c6a124" />

<br>

작성한 프리셋과 짝을 이루는 애니메이터를 작성. 해당 애니메이터에 프리셋을 참조시키고 키로 호출하여 `Play()` 매서드로 애니메이션을 실행함. 너무 복잡한 애니메이션은 어쩔수 없지만 `Event Trigger` 등과 활용 해 UI 에서 간편하게 비개발자 친화적으로도 애니메이션 적용 가능.
